# Zhuangwei Liu — Portfolio

Portfolio website for Zhuangwei Liu, a jewelry, accessory and product designer and contemporary artist.

## Development

```bash
pnpm install
pnpm dev
```

## Production build

```bash
pnpm build
```

The generated site is written to `dist/`.

## Deployment

The website is deployed through Cloudflare Pages from the `main` branch.

- Build command: `pnpm build`
- Build output directory: `dist`
- Node.js version: 20 or newer
